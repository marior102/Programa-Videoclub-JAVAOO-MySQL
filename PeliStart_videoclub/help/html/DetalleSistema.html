DETALLE DEL SISTEMA

Este sistema fue hecho a base de investigacion, como proceso de todo esto se daran cuenta que cada formulario esta vinculado con cada uno.

El sistema de video club tiene un formulario clientes que es para cargar y modificar, no se puede eliminar un cliente, ya que es mas o menos un club, donde los los clientes pueden aparecer como socios o normal. Tiene un catalogo donde se ven todas las peliculas disponible en el local y ejemplares, que es para agregar el stock de una pelicula.

En materia de movimiento del sistema, tenemos para registrar los alquileres, devolucion de alquileres y la venta de una pelicula, todos muy estructurado y con un interfaz muy facil de manejar.

En los reportes tenemos los registros de alquileres y la lista de pelicullas.
