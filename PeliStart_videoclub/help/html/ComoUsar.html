Â¿Como Usar?

Java Help es una libreria opcional de java que nos permite poner ventanas de ayuda a nuestras aplicaciones

Al apretar F1 en tu MDI que te aparezca la ventana de ayuda de Java, o mejor dicho que te salte esa ventana de ayuda..

En la parte derecha de la pantalla esta el texto de ayuda y la parte izquierda el panel de control

Tiene tambien un metodo Buscar, en caso de que quiera buscar un texto especifico

Esta ventana de ayuda es muy simple, pero es lo que pudimos hace, espero que les haya gustado.. Gracias..
