Version Netbeans 7.1.2

La Version de ID es la 7.1.2 que entre los cammbios implementados destaca el soporte para JavaFX2.0, soporte para WebLogic 12c, soprte para CSS3, mejoras en constructor de interfaces Swings, mejoras en el debug de PHP o integracion con Git, entre otras cosas.

Oracle libera con el lanzamiento de Netbeans ID 7.1 el primer entorno de desarrolo Java con soporte para JavaFX 2.0 con lo que es posible llevar cabo los procesos completos de compilacion y depuracion de las aplicaciones desarrolados en esta plataforma.
